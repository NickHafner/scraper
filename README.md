# Overview

This application is a terminal application to assist in web scraping with a main goal of being able to scrape Writing sites, index a particular work, scrape all existing chapter, extract the content and then convert that into an epub

## Getting Started

- Configuration found in `wails.json`; ref: <https://wails.io/docs/reference/project-config>
- Start dev with run `wails dev`. You can find the dev server running <http://localhost:34115>

## Feature Goals

1. Persistence (Search History, File Management, Dynamic Settings)
1. AI Assistance Mode (Ollama)
1. Chapter Index Scraping and follow up webpage retrieval
1. Sites with Login
